hexo个人博客搭建

Posted on 2017-04-12 | In hexo博客搭建 | | Visitors

###叨的叨叨以叨叨，那是什么叨
欢迎来到我的博客，该说啥呢。。。博客不是第一次写，但在自己搭建的博客上写的这第一篇博客确十分兴奋，这是为啥呢。哈哈，可能看着自己辛苦搭建出来的博客，一点一点达到自己心目中的样子，这种感觉可能只有做过的人才会体会到吧。如果你还在犹豫，那就别墨迹了。废话不多说。
搭建自己的博客，当时我查的时候好多人推荐用Wordpress，既然这么多人推荐，那我就不用，怎么滴。好吧被看穿了，其实是我太穷了，没钱租服务器。虽然好像也可以免费做出来，这个不太清楚，因为没太了解。就在我茶不思饭不想的时候，哎，怎么的，怎么可以这样，我发现了Hexo，柳暗花明的感觉好爽呀，这个真是太符合我了利用github来部署静态博客，谁叫咋就是干前端的呢。啥也不说了，立马开干。

###github

####注册github
github应该很多人已经注册或正在注册的路上。我就不多说啥了。

####创建页面仓库

选择New repository

Repository name输入：你的账号名字.github.io，如（cspp01.github.io）。最下面的选框Initialize this repository with a README可选可不选，建议选上。点击Create。好了，你在网上存放博客的地方也见好了。

####SSH密钥

#####生成SSH密钥
还是右键点击Git base Here输入

按回车，按回车，按回车，你得按3边，成功，到C:\Users\Administrator.ssh下，可以看到两个文件，id_rsa是密钥 ，id_rsa.pub是公钥。

#####建立连接
回到github选择setting->SSH and GPG keys->右上角New SSH key



title随便了，最好有意义点。打开刚才的id_rsa.pub，Ctrl+a->Ctrl+c，粘贴到key，Add SSH key，OK。
